# IEEE/ACM Monmouth University Chapter Website

Official website for the IEEE/ACM Monmouth University student chapter. A responsive web platform featuring Monmouth University branding, chapter information, activities overview, and engagement tools including membership signup, Discord community access, and social media links.

## 🌐 Live Website

[Visit our website](https://ieee-acm-monmouthuniversity.onrender.com)

## 📋 Features

- **Responsive Design**: Mobile-first approach with CSS Grid layouts
- **Executive Board**: Detailed member profiles with contact information
- **Social Integration**: Discord community and Instagram links
- **Membership Signup**: Direct integration with Microsoft Forms
- **Modern UI**: Professional design using Monmouth University brand colors
- **Accessibility**: WCAG compliant design with proper semantic HTML

## 🎨 Design

- **Color Palette**: Official Monmouth University brand colors
- **Typography**: Proxima Nova font family
- **Logos**: Official IEEE and ACM organizational logos
- **Responsive**: Optimized for all devices (320px - 1400px+)

## 🚀 Deployment

This website is deployed as a static site on [Render](https://render.com). Any changes pushed to the main branch will automatically trigger a new deployment.

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/ieee-acm-monmouthuniversity.git
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```
